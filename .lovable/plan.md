

# Исправление расположения кнопок режимов

## Проблема

Сейчас кнопки ("Κάρτες", "Κουίζ", "Εξέταση") используют `flex flex-wrap`, из-за чего третья кнопка переносится на вторую строку — выглядит неаккуратно.

## Решение

В файле **`src/pages/Learn.tsx`** (строки 65-77):

1. Вернуть `grid grid-cols-3 gap-2` вместо `flex flex-wrap gap-2` для равномерного распределения.
2. Добавить кнопкам `w-full` для заполнения ячейки.
3. Уменьшить внутренние отступы кнопок (`px-2`) и размер текста (`text-[11px]`), чтобы текст гарантированно помещался в ячейку.
4. Добавить `whitespace-nowrap` на span, чтобы текст не переносился внутри кнопки.

Результат: три одинаковые по ширине кнопки в одну строку, текст полностью виден.
