

# Обновление дизайна главной страницы

## Проблемы сейчас
- Все секции выглядят одинаково — один и тот же темный фон, одинаковые карточки
- Aurora-эффект едва заметен, фон почти полностью однотонный
- Stat-карточки маленькие и невыразительные
- Topic-карточки не используют свои яркие цвета достаточно
- Между секциями — пустое однотонное пространство, нет ритма
- Карточки режимов обучения слишком простые

## Что изменим

### 1. Hero — больше жизни
- Увеличить aurora-блобы (размер и opacity) чтобы фон "играл"
- Добавить тонкую сетку-паттерн (grid pattern) на фон hero для глубины
- Badge "Подготовка к гражданству" — добавить анимированную рамку (animated-border) для привлечения внимания
- CTA-кнопка — добавить пульсирующий glow-эффект

### 2. Stats — крупнее и выразительнее
- Числа сделать gradient-text (text-gradient-aurora) вместо обычного белого
- Увеличить размер чисел с text-3xl до text-4xl
- Иконки — добавить gradient-greek фон вместо liquid-glass-button

### 3. Topics — акцентировать цвета
- Добавить тонкую цветную полосу (accent bar) сверху каждой карточки — цвет соответствует теме (history=синий, culture=фиолет, laws=зеленый, geography=оранж)
- Иконки сделать крупнее (w-14 h-14 вместо w-12 h-12)
- Добавить слабый градиентный фон каждой карточки в цвет темы

### 4. Режимы обучения — добавить нумерацию и стрелки
- Добавить номер шага (01, 02, 03) крупным полупрозрачным текстом в углу каждой карточки
- Иконки — при наведении gradient-greek фон вместо primary

### 5. Секция между Topics и Modes — убрать лишний отступ
- Уменьшить `py-24` до `py-16` для более плотной компоновки

### 6. Footer CTA (для неавторизованных) — оставить как есть, уже яркий
