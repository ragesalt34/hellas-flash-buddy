
## Проблема

На скриншоте видно, что экзаменационный блок (`lp-exam`) обрезается снизу / находится слишком близко к карточкам тем и к нижней границе контейнера. Это из-за `padding: 28px 16px 32px` на `.lp-wrap` и `gap: 14px` в `.lp-grid`.

### Что менять — одна строка CSS

**`src/pages/Learn.tsx`, строка 267:**

`.lp-grid` → увеличить `gap: 14px` до `gap: 20px` — чтобы между рядом карточек и экзаменационным блоком стало больше воздуха.

**Строка 255:**

`.lp-wrap` → `padding: 28px 16px 48px` — увеличить нижний отступ с `32px` до `48px`, чтобы экзаменационный блок не "прилипал" к низу.

```css
/* Было */
.lp-wrap { max-width: 1040px; margin: 0 auto; padding: 28px 16px 32px; }
.lp-grid { display: grid; grid-template-columns: 1fr 1fr; gap: 14px; }

/* Станет */
.lp-wrap { max-width: 1040px; margin: 0 auto; padding: 28px 16px 48px; }
.lp-grid { display: grid; grid-template-columns: 1fr 1fr; gap: 20px; }
```

Минимальное изменение, максимальный визуальный эффект — блок экзамена получит больше "воздуха" от карточек выше и от края страницы снизу.
