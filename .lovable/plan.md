

# Улучшить контраст карточек на тёмном фоне

## Проблема

Фон (`--background`) имеет lightness 5%, а карточки (`--card`) — 10%. Разница всего 5% — карточки сливаются с фоном.

## Решение

Увеличить lightness карточек и усилить их фоновое свечение через CSS.

### 1. `src/index.css` — переменные `:root`

| Переменная | Сейчас | Станет |
|---|---|---|
| `--card` | `222 42% 10%` | `222 38% 13%` |
| `--popover` | `222 40% 12%` | `222 38% 15%` |

### 2. `src/index.css` — переменные `.dark`

| Переменная | Сейчас | Станет |
|---|---|---|
| `--card` | `222 42% 9%` | `222 38% 12%` |
| `--popover` | `222 42% 11%` | `222 38% 14%` |

### 3. `src/index.css` — `.liquid-glass-card`

Усилить box-shadow для лёгкого свечения:
- Было: `box-shadow: 0 4px 24px hsl(var(--primary) / 0.05)`
- Станет: `box-shadow: 0 4px 30px hsl(var(--primary) / 0.08), 0 0 0 0.5px hsl(var(--foreground) / 0.06)`

Добавление внутреннего `0.5px` кольца через box-shadow (не border) создаст едва заметную границу глубины без видимой "рамки".

Итого: карточки станут на 3% светлее + получат мягкое свечение, оставаясь в стиле Aurora Glass.
