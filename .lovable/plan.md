

# Переход на тёмную тему по умолчанию

## Суть
Сейчас `:root` — светлая тема (background ~97% lightness). Пользователь хочет тёмное оформление "из коробки". Самый чистый путь — сделать дефолтные CSS-переменные тёмными, а aurora-эффекты, glass-карточки и градиенты оставить как есть — они уже хорошо работают на тёмном фоне.

## Что меняется

### 1. `src/index.css` — переменные в `:root`
Заменяем светлую палитру на тёмную (по сути, `:root` получает значения, близкие к текущему `.dark`):

| Переменная | Было | Станет |
|---|---|---|
| `--background` | `220 20% 97%` | `222 47% 7%` |
| `--foreground` | `222 47% 11%` | `220 20% 93%` |
| `--card` | `220 25% 99%` | `222 40% 11%` |
| `--card-foreground` | `222 47% 11%` | `220 20% 93%` |
| `--popover` | `220 20% 96%` | `222 40% 13%` |
| `--popover-foreground` | `222 47% 11%` | `220 20% 93%` |
| `--primary` | `237 73% 58%` | `234 85% 68%` (ярче для контраста на тёмном) |
| `--primary-foreground` | `225 100% 96%` | `240 40% 13%` |
| `--secondary` | `220 14% 46%` | `220 14% 35%` |
| `--secondary-foreground` | `0 0% 98%` | `220 20% 93%` |
| `--muted` | `220 14% 75%` | `220 14% 25%` |
| `--muted-foreground` | `220 9% 46%` | `220 14% 65%` |
| `--accent` | `250 100% 97%` | `261 60% 16%` |
| `--accent-foreground` | `258 89% 66%` | `255 91% 76%` |
| `--border` | `220 13% 88%` | `222 25% 16%` |
| `--input` | `220 13% 88%` | `222 25% 16%` |

| shadow vars | light | darker (усиленная непрозрачность) |

Аналогично для sidebar-переменных.

### 2. `src/index.css` — glass-утилиты
Слегка уменьшить прозрачность glass-карточек, чтобы на тёмном фоне они были видимы:
- `.liquid-glass` — `card / 0.5` -> `card / 0.4`, внутренние border-свечения чуть ярче
- `.liquid-glass-card` — `card / 0.65` -> `card / 0.5`
- `.liquid-glass-button` — `primary / 0.1` -> `primary / 0.12`

### 3. `src/index.css` — aurora-blob
Увеличить opacity блобов (на тёмном фоне они эффектнее):
- `.aurora-blob` — `primary / 0.25` -> `primary / 0.18`, но убрать избыточный blur

### 4. `src/pages/Index.tsx` — CTA-секция
Сейчас CTA-блок использует `gradient-greek opacity-90` — на тёмном фоне это нормально, не трогаем.

### 5. `src/components/layout/Header.tsx`
Без изменений — header уже использует `liquid-glass`, которая адаптируется через переменные.

### 6. Страницы Login/Register
Без изменений — они используют `liquid-glass-card` и `aurora-blob`, адаптируются автоматически.

## Файлы для изменения
- `src/index.css` — обновление `:root` переменных + тюнинг glass-утилит

## Что НЕ меняется
- Компоненты React (классы остаются)
- Tailwind config
- Логика, роутинг, flashcard fix
- Структура `.dark` (оставляем на случай если в будущем понадобится переключатель)
