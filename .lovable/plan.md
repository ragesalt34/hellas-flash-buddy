

# Улучшение контраста карточек на тёмном фоне

## Проблема
Сейчас `--background: 222 47% 7%` и `--card: 222 40% 11%` — разница в яркости всего 4 пункта. Карточки визуально сливаются с фоном. Border у карточек тоже слишком прозрачный (`primary / 0.08`).

## Решение
Увеличить разрыв между фоном и карточками + усилить видимость границ:

### 1. CSS-переменные (`:root` в `src/index.css`)

| Переменная | Было | Станет | Зачем |
|---|---|---|---|
| `--background` | `222 47% 7%` | `225 44% 5%` | Чуть темнее фон |
| `--card` | `222 40% 11%` | `222 40% 13%` | Карточка светлее — разрыв 8% вместо 4% |
| `--popover` | `222 40% 13%` | `222 38% 15%` | Popover ещё светлее для иерархии |
| `--border` | `222 25% 16%` | `222 20% 20%` | Заметнее граница |
| `--input` | `222 25% 16%` | `222 20% 20%` | Аналогично |
| `--muted` | `220 14% 25%` | `220 14% 22%` | Чуть темнее для контраста с text |

### 2. Glass-утилиты (`.liquid-glass-card`)
- Увеличить непрозрачность фона: `card / 0.5` на `card / 0.7`, `card / 0.3` на `card / 0.5`
- Усилить border: `primary / 0.08` на `primary / 0.15`
- Усилить inset glow: `100% / 0.04` на `100% / 0.07`

### 3. Glass-утилиты (`.liquid-glass`)
- Фон: `card / 0.4` на `card / 0.6`
- Border: `primary / 0.1` на `primary / 0.15`

### 4. Glass-утилиты (`.liquid-glass-button`)
- Border: `primary / 0.12` на `primary / 0.18`

### 5. Flashcard glass
- Аналогичные увеличения непрозрачности

## Итог
Карточки будут чётко выделяться на фоне: более светлый card, более заметные border, сохраняя при этом стеклянную эстетику Aurora Glass.

## Файл для изменения
- `src/index.css` — переменные `:root` и glass-утилиты
