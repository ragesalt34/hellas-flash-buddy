
# Исправление обрезки текста кнопок режимов

## Проблема

При `grid-cols-3` кнопки режимов ("Карточки", "Тест", "Экзамен") обрезаются до "Кар...", "Экз..." — не хватает ширины.

## Решение

В файле **`src/pages/Learn.tsx`** (строка 65-74) изменить компоновку кнопок:

1. Заменить `grid grid-cols-3` на `flex flex-wrap gap-2` — кнопки будут занимать ровно столько места, сколько нужно тексту.
2. Убрать `w-full` у кнопки, чтобы она не растягивалась на всю ячейку.
3. Убрать `truncate` у `<span>`, чтобы текст не обрезался.
4. Убрать `justify-start` — кнопки будут компактными с центрированным содержимым.

Результат: три аккуратные кнопки в ряд, текст виден полностью ("Карточки", "Тест", "Экзамен").
