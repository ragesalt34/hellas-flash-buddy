

# Добавить hover-описание на карточки тем

## Что сделаем
При наведении мыши на карточку темы — плавно появляется развернутое описание (что именно изучите). При уведении мыши — описание плавно исчезает.

## Реализация

### Файл: `src/pages/Index.tsx`

**1. Добавить подробные описания для каждой темы:**

В массив `topics` добавить поле `details` с кратким перечнем того, что изучит пользователь:
- **История Греции**: "Древняя Эллада и Античность, Византийская империя, Османский период, Война за независимость, Современная Греция и ЕС"
- **Культура и традиции**: "Национальные праздники, Православные традиции, Греческая кухня, Музыка и танцы, Символы и обычаи"
- **Законы и политика**: "Конституция Греции, Права и обязанности граждан, Структура правительства, Избирательная система, Процесс натурализации"
- **География Греции**: "Регионы и префектуры, Крупные города, Острова и архипелаги, Горы и реки, Климат и природа"

**2. Обновить компонент `TopicCard`:**

- Добавить скрытый блок с `details`, который появляется при hover
- Использовать CSS transition: `max-height`, `opacity` — плавное раскрытие
- Блок появляется под основным описанием
- Текст — список из 4-5 пунктов в виде маркированного списка с точками
- Стилизация: `text-muted-foreground/80`, мелкий шрифт (`text-xs`)

**3. Механизм показа/скрытия:**

Использовать группу (`group`) + CSS-переменные Tailwind:
- По умолчанию: `max-h-0 opacity-0 overflow-hidden`
- При hover (`group-hover:`): `max-h-40 opacity-100`
- Transition: `transition-all duration-500 ease-out`

Никакого state не нужно — чисто CSS через `group-hover`.

## Технические детали

Один файл: **`src/pages/Index.tsx`**
- Добавить `details: string[]` в каждый объект topics
- В `TopicCard` добавить div с маркированным списком, скрытый по умолчанию и видимый при group-hover
- Никаких дополнительных зависимостей или CSS-изменений не требуется
