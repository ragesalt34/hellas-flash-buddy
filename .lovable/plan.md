
## Redesign страницы "Темы" (/learn) — 2026 тренды

### Что сейчас
- 2-колоночная сетка из стеклянных карточек с цветной полоской сверху
- Иконка + круговой прогресс + название + описание + 2 кнопки
- Тёмная карточка экзамена внизу
- Визуально монотонно, нет иерархии, нет движения, карточки одинакового размера

### 2026 тренды для применения
1. **Bento Grid** — карточки разного размера (featured card большая, остальные компактнее)
2. **Organic shapes** — blob-декораторы как в GLP-1 дизайне, уже используемые в проекте
3. **Ambient color zones** — каждая карточка имеет свой цветной ambient glow снизу/за собой
4. **Progress as hero** — прогресс-кольцо крупное, по центру карточки
5. **Micro-interactions** — hover поднимает карточку + усиливает glow, active-tap scale(0.97)
6. **Tactile feel** — тиснёный inset border + specular top highlight (уже в glass системе)
7. **Content-first typography** — серифные цифры прогресса (EB Garamond уже подключён)

### Конкретные изменения

**`src/pages/Learn.tsx`** — полная переработка разметки:

1. **Bento grid**: `grid-template-columns: 1.4fr 1fr` на десктопе. История занимает `row-span: 2` — самая большая карточка как "featured". Остальные компактнее.
2. **Крупное прогресс-кольцо**: в featured-карточке кольцо 72px, в обычных 44px. Процент отображается серифным шрифтом (`font-serif`, `EB Garamond`) внутри кольца.
3. **Ambient glow**: псевдо-элемент `::after` каждой карточки с `radial-gradient` в цвете темы, `filter: blur(40px)`, `z-index: -1`, opacity нарастает при hover.
4. **Blob decoration**: SVG-блоб в правом верхнем углу карточки с `opacity: 0.08`, как в GLP-1 дизайне.
5. **Кнопки**: чуть крупнее, `border-radius: 14px` вместо pill — более современно, с цветным border в цвете темы при hover.
6. **Экзамен** — переезжает в нижний правый сектор bento-сетки, занимает оставшееся место горизонтально через `col-span`.
7. **Стеклянный header страницы**: крупная надпись с `EB Garamond`, суб-текст светлее.

**`src/index.css`** — добавить:
- `.learn-bento-grid` — новый grid с адаптивными размерами
- `.learn-topic-card--featured` — увеличенный padding и размер кольца
- `.learn-topic-card--glow` — ambient glow через ::after
- `@media (max-width: 639px)` — 1 колонка, все карточки обычного размера

### Только `Learn.tsx` и `src/index.css`
Нет изменений в других компонентах, данные и API остаются прежними.
